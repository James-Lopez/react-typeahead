# typeahead
Application for autocompleting search terms.

## Usage
```bash
$ mongod
$ npm install
$ npm run seed
$ npm start
```

## Uses
* HTML5
* CSS3
* Modern JavaScript (ES2015+)
* MongoDB
* Express
* Node.js
